About data

The data of this website is from Our World in Data.

Our World in Data (OWID) is a scientific online publication that focuses on large global problems such as poverty, disease, hunger, climate change, war, existential risks, and inequality.

Our World in Data takes data from the European Center for Disease Prevention and Control (ECDC).

You can check out more details about the data on this page.

To be able to provide better information than the raw data this website has some conditional rules.

  • Sometimes a new cases value can be 0 as a temporary value. If the country had more 20 cases in the past week the table ignores the 0 and show the last valid data.
  • When a new cases value is minus for an adjustment, the table shows it as 0 in trend charts.
